## The Kernel vs The Shell

### What's the Kernel?

The kernel is the core, centremost part of the computer. Think of it as the part where all the decisions take place. 

### What's the Shell?

The shell, also known as the terminal in colloquial terms, is the opposite of the kernel. It sits on the outer layer, facing the user and translating input into instructions the kernel can understand.

### What's Linux's shell?

The shell for Linux is BASH, abbreviated from 'Bourne-Again Shell'. However, other shells can be used, such as ZSH, which are similar, but I'll get into more depth regarding that further on. Bash features several commands, which allow us to create, delete, move, rename and edit files and directories.
